  • 安装hexo

    直接命令 简单快捷：npm install -g hexo-cli

    修改命令为=> npm install -g hexo-cli –unsafe-perm

    然后创建一个respository 注册名.github.io 必须这样命名 例如我的是agclyy.github.io
