搭建blog几周后，重新写的时候报错

> ```
> WARN  No layout: index.html
> ```

原因是主题文件没有了，需要重新clone下了

我这边用的主题是BlueLake

blog文件夹下执行命令：

```
git clone https://github.com/chaooo/hexo-theme-BlueLake.git themes/BlueLake
```
